# Developer Documentation Index

Use this index to navigate the project documentation by folder/module.

Core docs:
- `docs/ARCHITECTURE.md`: high-level architecture and runtime flows.
- `docs/protocol.md`: UDP protocol specification (binary).
- `docs/comparative_study.md`: technology choices and trade-offs.
- `docs/STRUCTURE.md`: top-level layout overview.

Folder/module guides:
- `engine/core/README.md`: ECS core (entity, registry, sparse_array).
- `engine/game/README.md`: gameplay components and systems.
- `engine/net/README.md`: networking helpers and serialization.
- `engine/render/README.md`: rendering helpers (SFML integration).
- `engine/audio/README.md`: audio abstraction (`IAudio`).
- `server/README.md`: authoritative server app and loop.
- `client/README.md`: graphical client app, input, snapshot apply, HUD/UI.
- `scripts/README.md`: helper scripts (vcpkg setup, CI helpers).
- `cmake/README.md`: CMake structure and presets.
- `config/README.md`: config folder purpose and conventions.
- `sprites/README.md`: sprites/assets guidelines.
- `testing/README.md`: tests and how to run them.
- `docs/README.md`: documentation workflow (including optional Doxygen).
